# End-User / Technical Support Labs Portfolio

This repository contains hands-on labs and troubleshooting runbooks I practiced while preparing for an End-User / Technical Support role.

## What you’ll find here
- Realistic **ticket-style labs** (issue → investigation → fix → verification → documentation)
- **Runbooks** for common Tier 1/Tier 2 scenarios
- Evidence (screenshots/logs) when available

## Lab Categories
- **Operating Systems:** Windows / Linux / macOS
- **Networking & Connectivity**
- **Active Directory & User Management**
- **Remote Support & Troubleshooting**
- **Hardware & Peripherals (POS, printers)**
- **Security & Compliance**
- **Cloud Fundamentals (AWS)**
- **Virtualization & VMs**

## How I document each lab
Each lab includes:
- Goal
- Tools / environment
- Scenario (ticket style)
- Steps (runbook)
- Verification
- Troubleshooting notes
- What I learned
- Evidence (optional)
